# ガチャ期待値＆爆死シミュレーター

Dash を使って「SSRが出るまでに何回ガチャを回すことになるのか？」を
確率シミュレーションで可視化する Web アプリです。

排出率やガチャ1回の値段を設定して、
「平均でいくら溶けるのか」「◯回回しても出ない確率」を視覚的に確認できます。

---

## 🔥 このアプリでできること

- SSR排出率の自由な設定（%）
- 1回あたりの課金額を指定
- シミュレーション回数の変更
- 「◯回引いても当たらない」爆死確率の算出
- SSRが出るまでに必要な回数の分布をヒストグラムで表示
- 次の統計を自動計算：
  - 平均回数・平均課金額
  - 中央値
  - 90%の人が当たるまでに必要な回数（90パーセンタイル）
  - 爆死確率

---

## 🛠 使用技術

- Python 3.10+
- Dash
- Plotly
- NumPy
- uv（本プロジェクトでは軽量なパッケージ管理／実行ツールとして uv を使用しています）

---

## 🚀 セットアップ手順

### 1. プロジェクト作成

```bash
mkdir gacha-simulator
cd gacha-simulator
uv init
```

### 2. 依存関係インストール

```bash
uv add dash plotly numpy
```

---

▶️ 起動方法

```bash
uv run main.py
```

起動後、ブラウザで以下にアクセス：

http://127.0.0.1:8050/

---

📊 統計の見方

| 表示項目   | 意味                                     |
|------------|------------------------------------------|
| 平均回数   | 理論上の期待値（幾何分布の期待値）       |
| 中央値     | 半分の人がこの回数までに当たる回数       |
| 90%点      | 90% の人が当たるまでの回数（90パーセンタイル） |
| 爆死確率   | 指定回数引いても出ない確率               |

爆死確率は次の式で計算しています：

`(1 - 排出率)^回数`

---

🧠 技術メモ

- SSRが出るまでの試行回数は「幾何分布」に従います。
- NumPy の `np.random.geometric()` を利用しています。
- 極端に大きな値（長い尻尾）は可視化のために 99 パーセンタイルでカットしています。
